Палиндром —– это строка, которая одинаково читается как слева направо, так и справа налево.

Из данной строки 
s
s путём удаления и перестановки букв надо получить палиндром максимальной длины. Среди всех таких палиндромов надо получить лексикографически минимальный. Количество удалений и перестановок символов может быть любым.
