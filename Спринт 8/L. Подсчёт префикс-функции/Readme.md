В этой задаче вам необходимо посчитать префикс-функцию для заданной строки.

Формат ввода
На вход подаётся строка, состоящая из строчных латинских букв. Длина строки не превосходит 
1
0
6
10 
6
 .

Формат вывода
Если длина входной строки 
L
L, то выведите через пробел 
L
L целых неотрицательных чисел —– массив значений префикс-функции исходной строки.

Ввод
Вывод
abracadabra
0 0 0 1 0 1 0 1 2 3 4

Ввод
Вывод

Вывод
